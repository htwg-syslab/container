## Installation von Docker Desktop

1. Besuchen Sie die [Docker-Website](https://www.docker.com/) und laden Sie die für Ihr **Betriebssystem** (Windows, macOS, Linux) geeignete Version von **Docker Desktop** herunter. Achten Sie bei macOS darauf, die passende Version für Ihren Prozessor ([Intel](https://www.intel.com/) oder [ARM](https://www.arm.com/)) auszuwählen.

2. Installieren Sie Docker Desktop. Auf Windows kann es erforderlich sein, ein Update für das **Windows Subsystem for Linux (WSL)** zu installieren. Docker weist Sie ggf. darauf hin. Weitere Informationen: [Microsoft WSL-Dokumentation](https://docs.microsoft.com/en-us/windows/wsl/).

3. **Starten Sie Docker Desktop** und öffnen Sie die Container-Übersicht über das Symbol oben links im Dashboard.
