## Pocketlab Einrichtung

Dieses Kapitel beschreibt die Einrichtungsschritte, die für alle Pocketlab-Container gelten — unabhängig davon, ob Sie das **BSYS**- oder **ESYS**-Labor besuchen.

Starten Sie zuerst Ihren labspezifischen Container:

- **BSYS-Labor:** Siehe [AIN BSYS — bsyslab Container](BSYS-Container.md)
- **ESYS-Labor:** Siehe [ESYS — esyslab Container](ESYS-Container.md)

Anschließend folgen Sie den Schritten in diesem Kapitel für Anmeldung, VSCode-Integration und weitere Konfiguration.

### Übersicht labspezifische Werte

Verwenden Sie in den folgenden Anleitungen die Werte Ihres Labors:

| | BSYS | ESYS |
|---|---|---|
| Container-Name | `bsyslab` | `esyslab` |
| SSH-Port | `40405` | `40407` |
| SSH-Config Host | `bsyslab` | `esyslab` |
| Key-Datei | `id_rsa_bsyslab.key` | `id_rsa_esyslab.key` |
